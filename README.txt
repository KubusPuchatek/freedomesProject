Projekt stworzony na potrzeby zadania rekrutacyjnego dla firmy FREEDOMES.

Projekt spełnia następujące założenia:
- system pracuje jako aplikacja webowa
- dostęp do funkcjonalności skracania URL mają jedynie autoryzowani użytkownicy
- adres URL tłumaczony jest na krótki, losowy ciąg znaków
- możliwym ma być dodanie własnego ciągu znaków bez konieczności jego losowania
- wywołanie skróconego adresu powinno zostać odnotowane (odnotowywana jest jedynie ostatnie wywołanie)
